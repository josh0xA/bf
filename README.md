# bf - A Simplistic Interpreter For The Brainfuck Programming Language (beta)

## Installation 
a) ``git clone https://github.com/josh0xA/bf.git``<br/>
b) ``cd bf``<br/>
c) ``make``<br/>
d) ``./bf HelloWorld.bf``<br/>
